### Step 1: Extract Template from LLM Commander
```bash
# Extract all components (default)
python scripts/extract_template.py

# Or extract specific components
python scripts/extract_template.py --components rag_system orchestration configuration

# Custom output location
python scripts/extract_template.py --output my_template
```

### Step 2: Deploy to Target Repository  
```bash
# Deploy to current directory
python scripts/deploy_template.py /path/to/other/repo

# Dry run first (preview changes)
python scripts/deploy_template.py /path/to/other/repo --dry-run

# Force deployment (overwrite existing)
python scripts/deploy_template.py /path/to/other/repo --force
```

### Step 3: Configure in Target Repository
```bash
cd /path/to/other/repo

# Edit the configuration
cp llmc/config/local.example.toml llmc/config/local.toml
# Edit local.toml with your settings

# Test the RAG system
llmc/scripts/rag_refresh.sh
llmc/scripts/claude_wrap.sh "test query"

# Validate configuration  
python -m config.cli validate
```
