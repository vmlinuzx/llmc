# llmc-rag-repo Reference

Generated from `tools.rag_repo.cli --help`

```text
LLMC RAG Repo Tool

Manage which repos are tracked by the LLMC RAG daemon.

Usage:
  llmc-rag-repo <command> [options]

Commands:
  add        Register a repo and create its .llmc/rag workspace
  remove     Unregister a repo
  list       List all registered repos
  inspect    Show detailed info for a single repo
  help       Show this help overview

Examples:
  llmc-rag-repo add /home/you/src/llmc
  llmc-rag-repo list
  llmc-rag-repo inspect /home/you/src/llmc

```
